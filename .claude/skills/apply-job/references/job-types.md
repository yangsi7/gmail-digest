## Classification Algorithm

```
1. Scan JD for keywords from each type
2. Count matches per type
3. Select type with most matches
4. If tie, prefer in order: A > B > C > D
5. Confirm with user before proceeding
```

---

## Hybrid Roles

Some roles span multiple types. Handle as follows:

| Combination | Primary Type | Secondary Emphasis |
|-------------|--------------|-------------------|
| Health + AI | Type A | Include ML/analytics from Type B |
| AI + DevTools | Type C | Include ML expertise from Type B |
| FinTech + AI | Type D | Include AI/ML from Type B |
| Health + Sustainability | Type A | Include climate background from Type D |
